# Glossary

## SceneGraph

The CPU hierarchy of nodes, transforms, visibility, layers, and object IDs.

## Geometry

CPU vertex and index data. It is not a GPU buffer until registered with the renderer.

## MaterialId

A stable ID used by scene nodes and renderer material stores.

## BVH

Bounding volume hierarchy used by `scenix-raycaster` to reduce candidate mesh tests.

## LineGeometry

Debug line data generated by `scenix-helpers` for grids, axes, bounds, cameras, lights, arrows, and skeletons.

## PostStack

A sequence of GPU full-screen post-processing effects.

## Facade

The `scenix` crate, which re-exports focused crates behind stable feature flags.
