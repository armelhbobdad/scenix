# Physical Material

## Purpose

Exercises physical material preview fields.

## Source

`examples/physical_material.rs`

## Relevant Feature Flags

renderer

## Run Or Check

```sh
cargo run -p scenix --example physical_material --features renderer
```

## What To Look For

- The example should compile with the listed features.
- CPU examples should not require GPU setup.
- Renderer examples may need a working native graphics backend or headless support.
